Complete Single Room House Wiring Diagram

Warning: Electrical wiring can be dangerous. Always consult a licensed electrician for any electrical work in your home.

Wiring Method:

Step 1: Plan the Layout

Before starting, plan the locations of lights, switches, and power sockets. Ensure compliance with local electrical codes.

Step 2: Install the Distribution Board

Mount the distribution board (DB) near the main power entry point. This will house circuit breakers and act as the central point for all circuits.

Step 3: Run the Main Power Line

Connect the main power line to the distribution board. This should be done by a licensed electrician.

Step 4: Install the Lighting Circuit

Run wires from the DB to each light fixture location. Include switches in this circuit. Use yellow wires for easy identification.

Step 5: Install the Power Socket Circuit

Run a separate circuit from the DB to all power socket locations. Use blue wires for this circuit.

Step 6: Earth Wiring

Install earth wiring throughout the room, connecting to all metal fixtures and the earth bar in the DB. Use green wires for earthing.

Step 7: Connect Fixtures and Sockets

Install and connect all light fixtures, switches, and power sockets according to manufacturer instructions.

Step 8: Final Checks and Testing

Perform thorough checks of all connections. Test each circuit for proper functioning and safety.

Important: Always follow local electrical codes and regulations. It's highly recommended to have all electrical work inspected by a certified professional before use.
