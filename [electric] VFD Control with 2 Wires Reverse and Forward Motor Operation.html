Controlling VFD with 2 Wires: Reverse and Forward Motor Operation

Introduction

This document explains how to control a Variable Frequency Drive (VFD) using only two wires while enabling both forward and reverse motor operation. This method is efficient and simplifies wiring in many industrial applications.

Components Required

  • Variable Frequency Drive (VFD)
  • Three-phase AC motor
  • Two-wire control cable
  • SPDT (Single Pole Double Throw) switch
  • Potentiometer (for speed control, optional)

Wiring Method

  1. Connect the VFD to the power supply and the motor as per the manufacturer's instructions.
  2. Locate the control terminal block on the VFD.
  3. Identify the following terminals on the VFD:
    • Common (COM) terminal
    • Forward Run (FWD) terminal
    • Reverse Run (REV) terminal
  4. Connect one wire from the SPDT switch to the COM terminal.
  5. Connect the other two wires from the SPDT switch to the FWD and REV terminals respectively.

SPDT Switch Configuration

Switch Position Connection Motor Operation
Center Open Stop
Up COM to FWD Forward Rotation
Down COM to REV Reverse Rotation

VFD Configuration

Configure the VFD parameters for 2-wire control:

  1. Set the control mode to "2-wire control" or equivalent in the VFD settings.
  2. Configure the input terminals for forward and reverse run commands.
  3. Set acceleration and deceleration times as required for your application.

Note: The exact parameter names and setting procedures may vary depending on the VFD manufacturer. Consult your VFD's manual for specific instructions.

Operation

  1. To start the motor in forward direction, flip the SPDT switch to the "Up" position.
  2. To reverse the motor direction, flip the SPDT switch to the "Down" position.
  3. To stop the motor, return the SPDT switch to the center position.

Speed Control (Optional)

If speed control is required:

  1. Connect a potentiometer to the VFD's analog input terminals.
  2. Configure the VFD to accept speed reference from the analog input.
  3. Adjust the potentiometer to control the motor speed.

Safety Precautions

  • Always follow proper electrical safety procedures and local electrical codes.
  • Ensure the VFD and motor are properly grounded.
  • Implement appropriate safety interlocks and emergency stop mechanisms.
  • Verify that the motor can safely operate in both directions before applying reverse rotation.

Conclusion

This 2-wire control method provides a simple and effective way to operate a VFD-controlled motor in both forward and reverse directions. It reduces wiring complexity while maintaining full control over the motor's operation.
