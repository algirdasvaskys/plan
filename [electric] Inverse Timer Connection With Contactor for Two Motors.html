Inverse Timer Connection With Contactor for Two Motors

This document provides a comprehensive guide on setting up an inverse timer connection with a contactor for controlling two motors. This setup allows for alternating operation of two motors, where one motor runs while the other is stopped, and vice versa.

Components Required

  • Two three-phase motors
  • Two contactors (one for each motor)
  • Inverse timer (or alternating relay)
  • Overload relays (one for each motor)
  • Control circuit components (pushbuttons, indicator lights)
  • Power supply
  • Wiring and terminals

Step-by-Step Guide

Step 1: Power Circuit Setup

Connect the main power supply to the contactors. Each contactor should be wired to its respective motor. Ensure proper sizing of contactors and overload relays based on motor ratings.

Step 2: Control Circuit Wiring

Wire the control circuit, including the inverse timer, start/stop pushbuttons, and indicator lights. The inverse timer will alternate the operation between the two contactors.

Step 3: Inverse Timer Configuration

Configure the inverse timer to alternate between the two outputs at the desired interval. This will determine how long each motor runs before switching to the other.

Step 4: Contactor Coil Connections

Connect the outputs of the inverse timer to the coils of the respective contactors. When one output is energized, it will close its associated contactor, starting that motor.

Step 5: Overload Relay Integration

Integrate the overload relays into the control circuit. Wire them in series with the contactor coils to ensure motor protection.

Step 6: Testing and Adjustments

Test the system thoroughly. Verify that the motors alternate as intended and that all safety features (overload protection, emergency stop) function correctly. Make adjustments to the timer settings if necessary.

Safety Warning

Always ensure power is disconnected before working on electrical circuits. This installation should only be performed by qualified electrical technicians. Follow all relevant electrical codes and safety standards.

Operation

When the system is started, the inverse timer will energize one contactor, starting the first motor. After the set time interval, it will de-energize this contactor and energize the other, stopping the first motor and starting the second. This process will continue to alternate as long as the system is running.

Maintenance

Regular maintenance should include:

  • Checking all electrical connections for tightness
  • Inspecting contactors for wear and proper operation
  • Verifying the function of overload relays
  • Ensuring the inverse timer is operating correctly
  • Monitoring motor performance and checking for any unusual noise or vibration

By following this guide, you can set up an effective inverse timer connection with contactors for two motors, allowing for efficient alternating operation in various industrial applications.
