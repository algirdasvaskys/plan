Rectifier Type Instrument: Working Principle and Problem Solving

1. Introduction

Rectifier type instruments are widely used in electrical measurements to measure alternating current (AC) and voltage. These instruments combine the properties of both moving coil instruments and rectifiers to provide accurate measurements of AC quantities.

2. Working Principle

The working principle of a rectifier type instrument involves the following steps:

  1. Rectification of AC input
  2. Smoothing of rectified current
  3. Measurement using a moving coil meter

3. Components

A typical rectifier type instrument consists of:

  • Bridge rectifier
  • Moving coil meter movement
  • Resistors for current limiting and range selection
  • Capacitor for smoothing (in some designs)

4. Working Process

The working process can be broken down into these steps:

  1. AC input is applied to the instrument
  2. The bridge rectifier converts AC to pulsating DC
  3. The pulsating DC is smoothed (if a capacitor is present)
  4. The moving coil meter responds to the average value of the rectified current
  5. The scale is calibrated to show RMS values of sinusoidal AC input

5. Advantages

  • Can measure both AC and DC quantities
  • High sensitivity
  • Wide frequency range
  • Low power consumption

6. Limitations

  • Non-linear scale (crowded at lower end)
  • Affected by waveform distortions
  • Temperature sensitive
  • Frequency dependent (at higher frequencies)

7. Problem Solving Examples

Problem 1: Scale Calibration

A rectifier type ammeter has a full-scale deflection of 1 mA DC. What would be the full-scale AC current for sinusoidal input?

Solution 1:

For a rectifier type instrument with a bridge rectifier:

  • Average value of rectified sinusoidal current = 0.636 × RMS value
  • Moving coil responds to average value
  • Therefore, 1 mA DC = 0.636 × RMS value
  • RMS value = 1 mA / 0.636 = 1.57 mA

The full-scale AC current for sinusoidal input would be 1.57 mA RMS.

Problem 2: Frequency Response

A rectifier type voltmeter reads 100V at 50 Hz. At 1 kHz, it reads 98V for the same input voltage. Calculate the percentage error at 1 kHz.

Solution 2:

To calculate the percentage error:

  1. True value = 100V (assuming 50 Hz reading is correct)
  2. Measured value at 1 kHz = 98V
  3. Error = True value - Measured value = 100V - 98V = 2V
  4. Percentage error = (Error / True value) × 100 = (2V / 100V) × 100 = 2%

The percentage error at 1 kHz is 2%.

8. Conclusion

Rectifier type instruments are versatile tools for AC measurements, offering good sensitivity and a wide frequency range. Understanding their working principle and limitations is crucial for accurate measurements and problem-solving in electrical engineering applications.
