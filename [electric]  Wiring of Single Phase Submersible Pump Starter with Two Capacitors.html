Wiring of Single Phase Submersible Pump Starter with Two Capacitors

Warning: Electrical wiring can be dangerous. Always ensure that the power is turned off before working on any electrical connections. If you're not confident in your abilities, please consult a licensed electrician.

Components Required

  • Single Phase Submersible Pump Motor
  • Start Capacitor
  • Run Capacitor
  • Overload Relay
  • Contactor
  • Single Phase Power Supply
  • Circuit Breaker
  • Start/Stop Push Buttons
  • Wires

Wiring Procedure

  1. Install the circuit breaker at the main power supply.
  2. Connect the power supply to the input terminals of the contactor.
  3. Wire the start and stop push buttons to the contactor coil.
  4. Connect the overload relay to the contactor output.
  5. Wire the motor windings:
    • Connect the main winding to one output of the overload relay.
    • Connect the auxiliary winding to the other output of the overload relay.
    • Join the common point of both windings.
  6. Wire the capacitors:
    • Connect the run capacitor in parallel with the auxiliary winding.
    • Connect the start capacitor in series with the auxiliary winding and in parallel with the run capacitor.
  7. Connect the centrifugal switch (if present) in series with the start capacitor.
  8. Ensure all ground connections are properly made.
  9. Double-check all connections before powering on the circuit.

Operation

When the start button is pressed, the contactor closes, allowing current to flow through both the main and auxiliary windings. The start capacitor provides a high starting torque. As the motor reaches about 75% of its rated speed, the centrifugal switch (if present) opens, disconnecting the start capacitor. The motor continues to run with the main winding and the run capacitor connected to the auxiliary winding.

Maintenance

  • Regularly inspect all connections for signs of wear or damage.
  • Check capacitor condition periodically; replace if bulged or leaking.
  • Ensure the overload relay is set to the correct current rating for the motor.
  • Keep the starter box clean and dry to prevent short circuits.
Remember: Always follow local electrical codes and regulations when installing or modifying electrical equipment.
