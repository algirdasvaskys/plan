Star Delta Starter Control Circuit Diagram

Introduction

A Star Delta Starter is a motor starting method that reduces the start-up current for large AC motors. This method initially connects the motor windings in a star configuration to reduce the voltage, and then switches to a delta configuration for normal running operation.

Components

  • Main Contactor (K1)
  • Star Contactor (K2)
  • Delta Contactor (K3)
  • Overload Relay (F2)
  • Timer (K4)
  • Start and Stop Push Buttons

Working Method

  1. Initial State: All contactors are open, and the motor is not running.
  2. Start Button Press: When the start button is pressed, the main contactor K1 and star contactor K2 close.
  3. Star Connection: The motor windings are connected in a star configuration, reducing the voltage across each winding to 1/âˆš3 of the supply voltage.
  4. Timer Activation: Simultaneously, the timer K4 is activated.
  5. Acceleration: The motor accelerates with reduced torque and current.
  6. Timer Expiry: After a preset time (typically 5-10 seconds), the timer K4 triggers.
  7. Transition: The star contactor K2 opens, and after a brief delay, the delta contactor K3 closes.
  8. Delta Connection: The motor windings are now connected in a delta configuration, receiving full supply voltage.
  9. Normal Operation: The motor continues to run at full speed and torque.
  10. Overload Protection: The overload relay F2 monitors the current and trips if an overload occurs.
  11. Stopping: When the stop button is pressed, all contactors open, and the motor stops.

Advantages

  • Reduces initial starting current (typically to about 1/3 of direct online starting)
  • Reduces voltage dips in the power supply system
  • Provides smoother acceleration for the motor and connected load
  • Suitable for high inertia loads

Applications

Star Delta starters are commonly used in industrial settings for large motors (typically above 5 HP) where reduced starting current is necessary. Applications include:

  • Pumps
  • Compressors
  • Conveyor systems
  • Fans
  • Crushers
