How to Draw a Wiring Diagram and Turn it into a PLC Program

EPLAN Tutorial: Reverse Forward Starter

Step 1: Understanding the Reverse Forward Starter

A Reverse Forward Starter is a motor control circuit that allows an electric motor to run in both forward and reverse directions. It typically consists of:

  • Two contactors (one for forward, one for reverse)
  • Overload relay
  • Start and stop buttons
  • Forward and reverse selector switches

Step 2: Drawing the Wiring Diagram in EPLAN

  1. Open EPLAN Electric P8
  2. Create a new project
  3. Add a new page and select "Circuit diagram"
  4. Use the symbol library to add components:
    • Power supply (3-phase)
    • Motor
    • Contactors (KM1 for forward, KM2 for reverse)
    • Overload relay
    • Push buttons (Start, Stop)
    • Selector switch (Forward/Reverse)
  5. Connect the components according to the circuit logic
  6. Add wire numbers and component designations

Step 3: Converting the Wiring Diagram to a PLC Program

To convert the wiring diagram to a PLC program, we need to translate the hardwired logic into ladder logic. Here's a basic outline:

  1. Identify inputs:
    • Start button
    • Stop button
    • Forward selector
    • Reverse selector
    • Overload contact
  2. Identify outputs:
    • Forward contactor (KM1)
    • Reverse contactor (KM2)
  3. Create ladder logic:
    • Use normally closed stop button and overload contact in series
    • Add a seal-in (latching) circuit for the start button
    • Create separate rungs for forward and reverse operation
    • Implement interlocking to prevent simultaneous activation of forward and reverse contactors

Step 4: Implementing the PLC Program in EPLAN

EPLAN supports PLC programming through its "PLC Navigator" feature:

  1. Open PLC Navigator
  2. Create a new PLC
  3. Define I/O assignments based on your wiring diagram
  4. Use the ladder logic editor to create your program
  5. Implement the logic as outlined in Step 3
  6. Assign symbolic names to inputs and outputs for clarity

Step 5: Testing and Simulation

EPLAN offers simulation capabilities to test your PLC program:

  1. Use EPLAN's simulation feature to test the program logic
  2. Verify that the forward and reverse operations work as expected
  3. Check that the interlocking prevents simultaneous activation of both contactors
  4. Ensure the overload protection functions correctly

Important Notes:

  • Always follow safety guidelines and electrical codes when implementing motor control circuits.
  • Ensure proper interlocking to prevent short circuits in the motor windings.
  • Consider adding time delays between forward and reverse operations to allow the motor to come to a complete stop.
  • Implement emergency stop functionality in both the hardware circuit and PLC program.

Conclusion

By following this tutorial, you should now be able to draw a wiring diagram for a Reverse Forward Starter in EPLAN and convert it into a functional PLC program. Remember to thoroughly test your implementation and always prioritize safety in your designs.
