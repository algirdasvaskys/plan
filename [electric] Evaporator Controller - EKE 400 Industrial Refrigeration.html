Evaporator Controller - EKE 400 Industrial Refrigeration

1. Introduction

The EKE 400 is an advanced evaporator controller designed for industrial refrigeration systems. It offers precise control of evaporator superheat, ensuring optimal system performance and energy efficiency.

2. Key Features

  • Adaptive superheat control
  • Multiple refrigerant support
  • Modbus communication
  • Alarm management
  • Data logging capabilities

3. Installation Method

Step 1: Mounting

Mount the EKE 400 controller in a DIN rail inside a control panel. Ensure proper ventilation to prevent overheating.

Step 2: Wiring

Connect power supply, pressure transducers, temperature sensors, and electronic expansion valve according to the wiring diagram provided in the manual.

Step 3: Configuration

Use the controller's display and buttons or connect via Modbus to set up initial parameters such as refrigerant type, superheat setpoint, and alarm limits.

4. Operation Method

Step 1: System Start-up

Power on the controller and verify all sensors are reading correctly. The controller will begin regulating superheat automatically.

Step 2: Monitoring

Regularly check the superheat value and valve opening percentage on the controller display. Ensure values are within expected ranges.

Step 3: Adjustment

Fine-tune parameters if necessary, such as minimum stable superheat or PI controller settings, to optimize performance for specific operating conditions.

5. Maintenance

  • Regularly check and clean electrical connections
  • Verify sensor calibration annually
  • Update firmware when new versions are available
  • Back up controller settings periodically

6. Troubleshooting

Common Issue: Unstable Superheat

If superheat fluctuates excessively:

  1. Check for air in the system
  2. Verify proper sensor placement
  3. Adjust PI controller settings

Common Issue: Communication Errors

If Modbus communication fails:

  1. Check wiring and connections
  2. Verify Modbus settings (address, baud rate, parity)
  3. Use a Modbus testing tool to isolate the issue

7. Safety Considerations

  • Always disconnect power before working on electrical connections
  • Follow proper lockout/tagout procedures during maintenance
  • Wear appropriate personal protective equipment when working with refrigerants
  • Ensure proper ventilation in the equipment room

Note: This document provides a general overview of the EKE 400 evaporator controller. Always refer to the manufacturer's official documentation for specific instructions and safety guidelines.
