Smart Single Phase Wi-Fi Meter for Remote Light Control

Introduction

This guide will walk you through creating a Smart Single Phase Wi-Fi Meter that allows you to control your home lights from anywhere in the world using a smartphone or computer.

Components Needed

  • ESP8266 NodeMCU (Wi-Fi enabled microcontroller)
  • Relay module (5V)
  • ACS712 current sensor
  • ZMPT101B voltage sensor
  • Power supply module (3.3V and 5V outputs)
  • Jumper wires
  • Breadboard or PCB
  • Enclosure box

Setup Process

  1. Connect the components:
    • Connect the relay module to a digital pin on the ESP8266
    • Connect the ACS712 current sensor to an analog pin
    • Connect the ZMPT101B voltage sensor to another analog pin
    • Power the ESP8266 and sensors using the power supply module
  2. Program the ESP8266:
    • Install the Arduino IDE and necessary libraries (ESP8266, Blynk)
    • Write code to read voltage and current, calculate power, and control the relay
    • Implement Wi-Fi connectivity and integrate with a cloud service like Blynk
  3. Set up the cloud service:
    • Create a Blynk account and new project
    • Add buttons for on/off control and displays for voltage, current, and power readings
    • Get the authentication token for your project
  4. Finalize the hardware:
    • Transfer the circuit from breadboard to PCB if desired
    • Place the assembled circuit into the enclosure box
    • Connect the device to your home's electrical system (consult a professional electrician)

Usage Instructions

  1. Power on the Smart Wi-Fi Meter
  2. Ensure the device is connected to your home Wi-Fi network
  3. Open the Blynk app on your smartphone or access the web dashboard
  4. Use the on/off button to control your lights remotely
  5. Monitor real-time voltage, current, and power consumption
Note: Always prioritize safety when working with electrical systems. If you're not comfortable with electrical work, please consult a licensed electrician for installation.

Troubleshooting

  • If the device doesn't connect to Wi-Fi, check your network settings and ensure the correct credentials are provided in the code
  • If readings seem inaccurate, calibrate the voltage and current sensors using known references
  • For any persistent issues, check all connections and ensure the code is properly uploaded to the ESP8266
