Automatic Night Lamp with Motion Sensor

Introduction

This guide will walk you through the process of creating a night lamp that automatically turns on when it detects motion and turns off after a set period of inactivity. This project is perfect for hallways, bathrooms, or any area where you'd like convenient, hands-free lighting.

Components Needed

  • Arduino Uno or compatible microcontroller
  • PIR (Passive Infrared) motion sensor
  • LED or small lamp (5V-12V)
  • Resistor (220-330 ohms)
  • Jumper wires
  • Breadboard
  • Power supply (5V-12V, depending on your lamp)

Step-by-Step Guide

  1. Set up the Arduino

    Connect the Arduino to your computer using a USB cable. This will provide power and allow you to upload the code.

  2. Connect the PIR Sensor

    The PIR sensor typically has three pins:

    • VCC (Power): Connect to 5V on the Arduino
    • GND (Ground): Connect to GND on the Arduino
    • OUT (Signal): Connect to digital pin 2 on the Arduino

  3. Connect the LED or Lamp

    If using an LED:

    • Connect the longer leg (anode) to digital pin 13 on the Arduino through a 220-330 ohm resistor
    • Connect the shorter leg (cathode) to GND on the Arduino

    If using a lamp:

    • Use a transistor or relay to control the lamp, as most lamps require more current than the Arduino can provide directly

  4. Write and Upload the Arduino Code

    Use the Arduino IDE to write and upload the following code:

    
const int pirPin = 2;    // PIR sensor pin
const int ledPin = 13;   // LED pin
const int delayTime = 30000; // Time to keep the light on (30 seconds)

void setup() {
  pinMode(pirPin, INPUT);
  pinMode(ledPin, OUTPUT);
  Serial.begin(9600);
}

void loop() {
  if (digitalRead(pirPin) == HIGH) {
    digitalWrite(ledPin, HIGH);
    Serial.println("Motion detected! Light ON");
    delay(delayTime);
  } else {
    digitalWrite(ledPin, LOW);
    Serial.println("No motion. Light OFF");
  }
}

  5. Test the Circuit

    Once the code is uploaded, the lamp should turn on when motion is detected and turn off after 30 seconds of no motion.

  6. Adjust and Finalize

    You can adjust the delayTime variable in the code to change how long the light stays on after motion is detected. You may also need to adjust the sensitivity of your PIR sensor using its built-in potentiometers.

Safety Note

If you're using a lamp that requires high voltage (e.g., 120V AC), please consult with a licensed electrician. Working with high voltage can be dangerous and may require additional safety measures and components.

Conclusion

You now have a functional automatic night lamp that turns on with motion and off after a period of inactivity. This project can be expanded by adding features like light sensors to only activate at night, or by integrating it into a larger home automation system.
