Auto Change Over Switch (ATS) Using One Contactor

Introduction

An Auto Change Over Switch (ATS) is a crucial component in electrical systems where a backup power source is required. Typically, ATSes use multiple contactors, but this guide will explain how to create an ATS using only one contactor, which can be more cost-effective and simpler to implement.

Components Required

  • 1 x Three-pole contactor
  • 1 x Voltage monitoring relay
  • 1 x Timer relay
  • 2 x MCBs (Miniature Circuit Breakers)
  • Wires and terminals

Working Principle

The ATS with one contactor works on the principle of prioritizing the main power supply. When the main supply is available, the contactor remains de-energized, allowing the main supply to power the load. If the main supply fails, the contactor energizes, switching the load to the backup supply.

Setup and Configuration

  1. Connect the main power supply to the normally closed (NC) contacts of the contactor.
  2. Connect the backup power supply to the normally open (NO) contacts of the contactor.
  3. Wire the voltage monitoring relay to monitor the main power supply.
  4. Connect the output of the voltage monitoring relay to the timer relay.
  5. Wire the timer relay output to the contactor coil.
  6. Install MCBs on both the main and backup power supply lines for protection.

Operation

  1. When the main power supply is available, the voltage monitoring relay keeps the timer relay de-energized.
  2. If the main power fails, the voltage monitoring relay detects the failure and triggers the timer relay.
  3. After a preset delay (to avoid unnecessary switching due to brief power fluctuations), the timer relay energizes the contactor.
  4. The energized contactor switches the load from the main supply to the backup supply.
  5. When the main power is restored, the voltage monitoring relay de-energizes the timer relay, which in turn de-energizes the contactor, switching back to the main supply.

Note:

The delay in the timer relay is crucial to prevent rapid switching between power sources during brief power fluctuations. Typically, a delay of 3-5 seconds is sufficient, but this can be adjusted based on specific requirements.

Warning:

Working with electrical systems can be dangerous. Always ensure that the installation is carried out by a qualified electrician and complies with local electrical codes and regulations.

Advantages

  • Cost-effective: Uses only one contactor instead of two.
  • Simpler design: Fewer components mean easier installation and maintenance.
  • Space-saving: Requires less space in the electrical panel.
  • Lower power consumption: One contactor consumes less power than two.

Conclusion

Using a single contactor for an Auto Change Over Switch is an efficient and cost-effective solution for implementing backup power systems. While it may not be suitable for all applications, especially those requiring simultaneous connection to both power sources, it is an excellent option for many residential and small commercial setups.
