Resistor Color Code Guide

Introduction

The resistor color code is a system used to indicate the resistance value and tolerance of resistors. It consists of colored bands painted on the resistor body.

Color Code Chart

Color Digit Multiplier Tolerance
Black 0 ×1 Ω -
Brown 1 ×10 Ω ±1%
Red 2 ×100 Ω ±2%
Orange 3 ×1 kΩ -
Yellow 4 ×10 kΩ -
Green 5 ×100 kΩ ±0.5%
Blue 6 ×1 MΩ ±0.25%
Violet 7 ×10 MΩ ±0.1%
Gray 8 ×100 MΩ ±0.05%
White 9 ×1 GΩ -
Gold - ×0.1 Ω ±5%
Silver - ×0.01 Ω ±10%

How to Read Resistor Color Codes

  1. 4-Band Resistors:
    • 1st band: First digit
    • 2nd band: Second digit
    • 3rd band: Multiplier
    • 4th band: Tolerance
  2. 5-Band Resistors:
    • 1st band: First digit
    • 2nd band: Second digit
    • 3rd band: Third digit
    • 4th band: Multiplier
    • 5th band: Tolerance
  3. 6-Band Resistors:
    • 1st band: First digit
    • 2nd band: Second digit
    • 3rd band: Third digit
    • 4th band: Multiplier
    • 5th band: Tolerance
    • 6th band: Temperature coefficient

Example

Let's say we have a resistor with the following color bands: Yellow, Violet, Orange, Gold

  • Yellow (4) - First digit
  • Violet (7) - Second digit
  • Orange (×1 kΩ) - Multiplier
  • Gold (±5%) - Tolerance

The resistance value would be: 47 × 1,000 = 47,000 Ω or 47 kΩ ±5%

Tips for Remembering the Color Code

A popular mnemonic device to remember the color code order is:

Bad Boys Run Over Yellow Gardens Before Viewing Green Waves

This corresponds to the colors: Black, Brown, Red, Orange, Yellow, Green, Blue, Violet, Gray, White
