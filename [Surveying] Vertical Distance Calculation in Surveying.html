Vertical Distance Calculation By Total Station or Theodolite Surveying

Introduction

In surveying, calculating vertical distances is crucial for determining elevations and creating topographic maps. Total stations and theodolites are precision instruments used for these measurements. This document explains the method for calculating vertical distances using these instruments.

Equipment

  • Total Station or Theodolite
  • Tripod
  • Reflector prism (for total station)
  • Leveling rod (for theodolite)

Method

  1. Set up the instrument:
    • Place the total station or theodolite on a tripod
    • Level the instrument using its built-in bubble levels
  2. Measure the instrument height (IH):
    • Measure the vertical distance from the ground to the instrument's horizontal axis
  3. Sight the target:
    • For total station: Aim at the reflector prism
    • For theodolite: Aim at the leveling rod
  4. Record measurements:
    • Vertical angle (VA): The angle between the line of sight and the horizontal plane
    • Slope distance (SD): The direct distance from the instrument to the target
  5. Calculate vertical distance

Calculation

The vertical distance (VD) is calculated using trigonometry:

VD = SD * sin(VA)

Where:

  • VD = Vertical Distance
  • SD = Slope Distance
  • VA = Vertical Angle

Elevation Difference

To find the elevation difference between the instrument station and the target:

Elevation Difference = VD + IH - TH

Where:

  • IH = Instrument Height
  • TH = Target Height (height of prism or reading on leveling rod)
Note: If the vertical angle is measured as a zenith angle (angle from vertical), use the cosine function instead:
VD = SD * cos(ZA)
Where ZA is the zenith angle.

Example

Given:

  • Slope Distance (SD) = 100 meters
  • Vertical Angle (VA) = 30 degrees
  • Instrument Height (IH) = 1.5 meters
  • Target Height (TH) = 2.0 meters

Calculate:

  1. Vertical Distance:
    VD = 100 * sin(30Â°) = 50 meters
  2. Elevation Difference:
    Elevation Difference = 50 + 1.5 - 2.0 = 49.5 meters

Conclusion

Accurate vertical distance calculations are essential in surveying for determining elevations and creating topographic maps. By using a total station or theodolite and applying trigonometric principles, surveyors can precisely measure vertical distances and elevation differences between points.
