Programmable Digital Timer Connection in DOL Starter with Selector Switch

1. Introduction

This guide provides step-by-step instructions for connecting a programmable digital timer in a Direct-On-Line (DOL) starter circuit with a selector switch. This setup allows for both manual and timed operation of the motor.

2. Components Required

  • DOL Starter
  • Programmable Digital Timer
  • Selector Switch (3-position: Manual-Off-Auto)
  • Contactor
  • Overload Relay
  • MCB (Miniature Circuit Breaker)
  • Push Buttons (Start and Stop)
  • Indicator Lights
  • Control Transformer (if required)
  • Wiring and Terminals

3. Wiring Procedure

  1. Install the MCB at the incoming power supply.
  2. Connect the programmable digital timer to the control circuit.
  3. Wire the selector switch with three positions: Manual, Off, and Auto.
  4. Connect the contactor and overload relay as per the DOL starter configuration.
  5. Wire the start and stop push buttons.
  6. Connect indicator lights for power, run, and trip status.
  7. Integrate the timer into the control circuit.

4. Detailed Connection Steps

  1. Connect the incoming power supply to the MCB.
  2. From MCB, connect to the contactor's main contacts.
  3. Wire the overload relay in series with the contactor's main contacts.
  4. Connect the motor to the overload relay output.
  5. For the control circuit:
    • Connect one line of the control supply to the common terminal of the selector switch.
    • Wire the "Manual" position of the selector switch to the start push button.
    • Connect the "Auto" position to the timer's output contact.
    • Wire the start button and timer output in parallel to the contactor coil.
    • Connect the stop button in series with the parallel combination of start button and timer output.
    • Wire the overload relay's normally closed auxiliary contact in series with the control circuit.
  6. Connect the timer's power supply terminals to the control circuit power.
  7. Wire indicator lights:
    • Power On: Connect across the control supply.
    • Run: Wire in parallel with the contactor coil.
    • Trip: Connect to the overload relay's normally open auxiliary contact.

Note:

Ensure that all connections are tight and properly insulated. Use appropriate wire sizes for both main and control circuits.

5. Timer Programming

Program the digital timer according to the desired on and off cycles. Refer to the timer's manual for specific programming instructions.

6. Operation

  • Manual Mode: Use start and stop push buttons to control the motor.
  • Auto Mode: Motor operates based on the programmed timer settings.
  • Off Mode: Motor remains off regardless of timer or push button status.

Safety Warning:

Always ensure power is disconnected before working on electrical circuits. Follow all relevant safety procedures and local electrical codes. Only qualified personnel should perform these connections.

7. Testing and Troubleshooting

  1. After completing all connections, double-check all wiring before powering on.
  2. Test each mode of operation (Manual, Off, Auto) to ensure proper functioning.
  3. Verify that the overload protection is working by simulating an overload condition.
  4. Check that the timer is accurately controlling the motor in Auto mode.
  5. If any issues arise, systematically check each component and connection.
