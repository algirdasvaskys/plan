DOL Starter: Local, Remote, and DCS Start-Stop Connection

Introduction

A Direct-On-Line (DOL) starter is a common method used to start and stop electric motors. This document explains the local, remote, and DCS (Distributed Control System) start-stop connections for a DOL starter.

1. Local Start-Stop Connection

Local control allows the motor to be started and stopped directly at the motor location.

  • Start pushbutton (NO contact) connected in parallel with the hold-in contact of the contactor
  • Stop pushbutton (NC contact) connected in series with the control circuit
  • Overload relay contact in series with the control circuit

2. Remote Start-Stop Connection

Remote control enables starting and stopping the motor from a distant location.

  • Remote start pushbutton (NO contact) connected in parallel with the local start button
  • Remote stop pushbutton (NC contact) connected in series with the local stop button
  • Additional wiring to connect remote station to the starter panel

3. DCS Start-Stop Connection

DCS control allows for automated control of the motor through a centralized control system.

  • DCS start command connected to a relay with NO contact, wired in parallel with start buttons
  • DCS stop command connected to a relay with NC contact, wired in series with stop buttons
  • Feedback signals (e.g., run status, fault status) wired to DCS inputs
  • Local/Remote/DCS selector switch to choose control mode

Key Components

  • Contactor: Main power switching device
  • Overload relay: Protects motor from overload conditions
  • Control transformer: Provides control voltage (typically 110V or 24V)
  • Pushbuttons: For manual control
  • Selector switch: To choose between local, remote, and DCS control
  • Auxiliary contacts: For status indication and interlocking
Note: Always refer to the specific manufacturer's guidelines and local electrical codes when implementing DOL starter connections. Safety should be the top priority in all electrical installations.
