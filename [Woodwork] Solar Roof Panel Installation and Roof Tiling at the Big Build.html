Solar Roof Panel Installation and Roof Tiling at the Big Build

Note: This document outlines the method for installing solar roof panels and completing roof tiling at the Big Build project. Always follow local building codes and manufacturer's instructions.

1. Preparation

  1. Conduct a thorough roof inspection
  2. Ensure the roof structure can support the additional weight
  3. Obtain necessary permits and approvals
  4. Gather all required tools and materials
  5. Set up safety equipment (harnesses, scaffolding, etc.)

2. Roof Tiling

  1. Install underlayment (felt or synthetic)
  2. Install battens or strapping for tile support
  3. Begin laying tiles from the bottom edge of the roof
  4. Work upwards, overlapping tiles as per manufacturer's instructions
  5. Cut tiles as needed for a proper fit around vents, chimneys, etc.
  6. Install ridge caps along the peak of the roof

3. Solar Panel Installation

  1. Mark locations for mounting brackets
  2. Install flashing at mounting points to prevent leaks
  3. Secure mounting brackets to the roof structure
  4. Attach rails to the mounting brackets
  5. Install microinverters or power optimizers (if using)
  6. Mount solar panels onto the rails
  7. Connect panels to microinverters or string them together

4. Electrical Work

  1. Install junction boxes and conduit as needed
  2. Run wiring from panels to the inverter location
  3. Install the main inverter (if not using microinverters)
  4. Connect the system to the main electrical panel
  5. Install a production meter and/or net meter as required

5. Final Steps

  1. Perform a thorough system check
  2. Test all electrical connections
  3. Clean up the work area
  4. Conduct a final inspection
  5. Obtain final approval from local authorities
  6. Provide system documentation and warranty information to the client
Safety First: Always prioritize safety throughout the installation process. Use proper personal protective equipment and follow all safety guidelines.
