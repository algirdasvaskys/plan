Sequence DOL Starter Connection with Two Contactors and Add-on Block Timer

1. Introduction

A Sequence DOL (Direct On Line) Starter with two contactors and an add-on block timer is used to start two motors in a specific sequence. This setup ensures that the second motor starts only after the first motor has been running for a predetermined time.

2. Components Required

  • Two 3-phase motors
  • Two 3-pole contactors (K1 and K2)
  • Two overload relays (OL1 and OL2)
  • One add-on block timer (T1)
  • Start and Stop push buttons
  • Main circuit breaker or fuses
  • Control circuit fuse

3. Connection Procedure

  1. Connect the main power supply to the main circuit breaker or fuses.
  2. From the main circuit breaker, connect to the common points of both contactors K1 and K2.
  3. Connect the output of contactor K1 to the input of overload relay OL1, then to Motor 1.
  4. Connect the output of contactor K2 to the input of overload relay OL2, then to Motor 2.
  5. For the control circuit:
    • Connect one side of the control supply to the Start button, Stop button (NC), OL1 (NC), OL2 (NC), and K1 coil in series.
    • Connect the other side of K1 coil to the other side of the control supply.
    • Connect a normally open auxiliary contact of K1 in parallel with the Start button for self-holding.
    • Connect the add-on block timer T1 to K1. The timer should be energized when K1 is energized.
    • Connect the normally open contact of T1 to the K2 coil.

4. Operation Sequence

  1. When the Start button is pressed, contactor K1 is energized, starting Motor 1.
  2. The add-on block timer T1 starts counting when K1 is energized.
  3. After the preset time on T1 elapses, its contact closes, energizing contactor K2 and starting Motor 2.
  4. Both motors continue to run until the Stop button is pressed or an overload occurs.

Note: The add-on block timer T1 is typically mounted directly onto contactor K1, simplifying the wiring and reducing panel space requirements.

Warning: Always ensure that the power is disconnected before working on electrical connections. This work should only be performed by qualified electricians.

5. Troubleshooting

  • If Motor 1 doesn't start, check K1, OL1, and the control circuit connections.
  • If Motor 2 doesn't start after the delay, check K2, OL2, T1, and their connections.
  • If both motors start simultaneously, check the T1 timer settings and connections.
