Calculating Cable Size for Motors and Loads

1. Cable Size for Motors

To calculate the cable size for a motor, follow these steps:

  1. Determine the motor's full load current (FLC)
  2. Apply any necessary correction factors
  3. Calculate the required cable ampacity
  4. Select the appropriate cable size based on ampacity tables

1.1 Determine Full Load Current (FLC)

For a three-phase motor:

FLC = (Motor HP × 746) / (√3 × VL-L × PF × Eff)

Where:

  • FLC: Full Load Current (Amperes)
  • Motor HP: Motor Horsepower
  • 746: Conversion factor (1 HP = 746 Watts)
  • VL-L: Line-to-Line Voltage
  • PF: Power Factor (typically 0.8 to 0.9 for motors)
  • Eff: Motor Efficiency (typically 0.8 to 0.95)

1.2 Apply Correction Factors

Adjust the FLC based on various factors:

  • Temperature: If ambient temperature differs from 30°C (86°F)
  • Voltage Drop: Consider voltage drop over long distances
  • Starting Current: For motors with high starting current
  • Grouping: When multiple cables are bundled together

1.3 Calculate Required Cable Ampacity

The required cable ampacity is typically 125% of the FLC for continuous duty motors:

Required Ampacity = FLC × 1.25

1.4 Select Cable Size

Use ampacity tables to select the appropriate cable size based on the required ampacity and insulation type.

2. Cable Size for Loads

To calculate cable size for general loads:

  1. Determine the load current
  2. Apply correction factors
  3. Calculate voltage drop
  4. Select cable size based on ampacity and voltage drop

2.1 Determine Load Current

For a single-phase load:

I = P / (V × PF)

For a three-phase load:

I = P / (√3 × VL-L × PF)

Where:

  • I: Current (Amperes)
  • P: Power (Watts)
  • V: Voltage (Volts)
  • VL-L: Line-to-Line Voltage (for three-phase)
  • PF: Power Factor

2.2 Apply Correction Factors

Similar to motor calculations, apply correction factors for temperature, grouping, etc.

2.3 Calculate Voltage Drop

For single-phase:

Vd = 2 × I × R × L / 1000

For three-phase:

Vd = √3 × I × R × L / 1000

Where:

  • Vd: Voltage Drop (Volts)
  • I: Current (Amperes)
  • R: Cable Resistance (Ohms/km)
  • L: Cable Length (meters)

2.4 Select Cable Size

Choose a cable size that satisfies both the ampacity requirement and keeps the voltage drop within acceptable limits (typically 3-5%).

3. Cable Size for 3-Phase Motors

The process for calculating cable size for 3-phase motors is similar to the general motor calculation, with a few specific considerations:

3.1 Full Load Current (FLC) for 3-Phase Motors

Use the formula mentioned earlier for three-phase motors:

FLC = (Motor HP × 746) / (√3 × VL-L × PF × Eff)

3.2 Voltage Drop Calculation

For 3-phase motors, use the three-phase voltage drop formula:

Vd = √3 × I × R × L / 1000

3.3 Consider Starting Current

3-phase motors often have high starting currents. The typical starting current is 6-8 times the FLC. Ensure the cable can handle this momentary high current.

3.4 Power Factor and Efficiency

3-phase motors generally have better power factor and efficiency compared to single-phase motors. Use accurate values for your specific motor when calculating.

Conclusion

Calculating the correct cable size is crucial for the safe and efficient operation of motors and electrical loads. Always consult local electrical codes and standards, as requirements may vary by region. When in doubt, consult a qualified electrician or engineer.
