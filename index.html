Regex Data Extraction Simulation

Visual Basic Code

Dim pattern As String = "\d+"
Dim input As String = "The price is $10.99."
Dim matches As MatchCollection = Regex.Matches(input, pattern)
Dim total As Integer = 0
For Each match As Match In matches
    total += Convert.ToInt32(match.Value)
Next

Code Explanation

The Visual Basic code demonstrates how to use regular expressions to extract data from a string:

  • It defines a regex pattern (in this case, \d+ to match one or more digits).
  • It uses Regex.Matches() to find all matches in the input string.
  • It iterates through the matches, converts each to an integer, and adds them to a total.

In our simulation:

  • You can enter any text containing various types of data.
  • You can choose from 100 pre-defined regex patterns or enter your own.
  • The simulation extracts all matches based on the selected pattern.
  • It displays each match and its position in the input string.

This simulation expands on the original example by allowing custom input and a wide range of regex patterns, providing a more comprehensive and educational experience in working with regular expressions.
