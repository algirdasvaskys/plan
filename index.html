OSI Model - Open Systems Interconnection

7 Application Layer

Provides network services directly to end-users and applications.

Protocols: HTTP, FTP, SMTP, DNS, Telnet
6 Presentation Layer

Handles data formatting, encryption, and compression.

Functions: SSL/TLS, JPEG, MPEG, ASCII
5 Session Layer

Manages sessions between applications.

Protocols: NetBIOS, RPC, PAP
4 Transport Layer

Ensures reliable data delivery between end systems.

Protocols: TCP, UDP
Devices: Gateway
3 Network Layer

Handles logical addressing and routing.

Protocols: IP, ICMP, IGMP
Devices: Router
2 Data Link Layer

Provides node-to-node data delivery.

Protocols: Ethernet, PPP, Frame Relay
Devices: Switch, Bridge
1 Physical Layer

Transmits raw bit streams over physical medium.

Components: Cables, Signals, Binary transmission
Devices: Hub, Repeater, Cable
