Traffic Lights Project with Arduino Uno - Beginner Level

Introduction

This project will guide you through creating a simple traffic light system using an Arduino Uno. We'll use three LEDs (red, yellow, and green) to simulate traffic lights.

Components Required

  • Arduino Uno board
  • Breadboard
  • 3 LEDs (1 red, 1 yellow, 1 green)
  • 3 resistors (220 ohms)
  • Jumper wires
  • USB cable for Arduino

Circuit Diagram

Wiring Instructions

  1. Connect the long leg (anode) of the red LED to digital pin 2 on the Arduino.
  2. Connect the long leg (anode) of the yellow LED to digital pin 3 on the Arduino.
  3. Connect the long leg (anode) of the green LED to digital pin 4 on the Arduino.
  4. Connect the short leg (cathode) of each LED to a 220-ohm resistor.
  5. Connect the other end of each resistor to the Arduino's GND (ground) pin.

Arduino Code


// Traffic Light Simulation

// Define pin numbers
const int redPin = 2;
const int yellowPin = 3;
const int greenPin = 4;

void setup() {
  // Initialize the LED pins as outputs
  pinMode(redPin, OUTPUT);
  pinMode(yellowPin, OUTPUT);
  pinMode(greenPin, OUTPUT);
}

void loop() {
  // Red light
  digitalWrite(redPin, HIGH);
  digitalWrite(yellowPin, LOW);
  digitalWrite(greenPin, LOW);
  delay(5000);  // Stay red for 5 seconds

  // Green light
  digitalWrite(redPin, LOW);
  digitalWrite(yellowPin, LOW);
  digitalWrite(greenPin, HIGH);
  delay(5000);  // Stay green for 5 seconds

  // Yellow light
  digitalWrite(redPin, LOW);
  digitalWrite(yellowPin, HIGH);
  digitalWrite(greenPin, LOW);
  delay(2000);  // Stay yellow for 2 seconds
}

Instructions to Run the Project

  1. Connect your Arduino Uno to your computer using the USB cable.
  2. Open the Arduino IDE on your computer.
  3. Copy and paste the provided code into a new Arduino sketch.
  4. Select the correct board (Arduino Uno) and port from the Tools menu.
  5. Click the "Upload" button to compile and upload the code to your Arduino.
  6. Once uploaded, the traffic light sequence should start automatically.

Troubleshooting

  • If the LEDs don't light up, double-check your wiring connections.
  • Ensure that the LEDs are connected in the correct orientation (long leg to Arduino pin).
  • Verify that you've selected the correct board and port in the Arduino IDE.
  • If you're still having issues, try using a multimeter to test the continuity of your connections.

Conclusion

Congratulations! You've successfully created a simple traffic light system using an Arduino Uno. This project demonstrates basic digital output control and timing in Arduino programming. Feel free to modify the timings or add additional features to enhance your traffic light simulation.
