Home Improvement Guide: Smart Windows, Korniche Lanterns, and First Fix Preparations

1. Smart Windows

Smart windows, also known as electrochromic windows, can change their tint or opacity on demand. Here's how to install and use them:

Installation Process:

  • Measure the window opening accurately
  • Order custom-sized smart windows
  • Remove old windows and clean the opening
  • Install the smart window frame
  • Insert the smart glass pane
  • Connect the window to the power supply and control system
  • Test the window's functionality

Usage:

  • Use the provided remote control or smartphone app to adjust tint levels
  • Set schedules for automatic tinting based on time of day or sunlight intensity
  • Integrate with home automation systems for centralized control

2. Korniche Lanterns

Korniche lanterns are modern roof lanterns that provide natural light and a contemporary look. Here's the installation process:

  • Prepare the roof opening according to manufacturer specifications
  • Install the insulated upstand around the opening
  • Apply weatherproofing membrane to the upstand
  • Position and secure the lantern frame to the upstand
  • Install the glazing bars and rafters
  • Insert the glass panels, ensuring proper sealing
  • Add finishing touches like caps and trims
  • Conduct a water test to ensure proper installation and sealing

3. Preparing for Studwork

Before beginning studwork, follow these steps:

  • Create a detailed floor plan and wall layout
  • Mark stud locations on the floor and ceiling
  • Ensure the floor is level and the ceiling is straight
  • Gather necessary materials: timber, fasteners, insulation
  • Cut bottom and top plates to size
  • Pre-drill holes for electrical and plumbing runs

4. First Fix Electrics

First fix electrical work involves installing wiring and boxes before walls are closed up. Steps include:

  • Plan the electrical layout, including socket and light switch positions
  • Install a temporary power supply for tools
  • Run cables through joists and studs, using proper cable protection where needed
  • Install back boxes for sockets and switches
  • Position junction boxes and consumer units
  • Label all cables clearly for second fix work
  • Conduct initial safety tests

5. First Fix Plumbing

First fix plumbing involves installing pipes and drainage before floors and walls are finished. Key steps:

  • Plan the plumbing layout, including water supply and waste pipe routes
  • Install main water supply pipes
  • Fit waste pipes and ensure proper fall for drainage
  • Position pipe clips and supports
  • Install soil stacks and vent pipes
  • Pressure test all pipework
  • Insulate pipes where necessary
  • Mark pipe locations for future reference

Remember to always consult with professionals and adhere to local building codes and regulations when undertaking these home improvement projects.
