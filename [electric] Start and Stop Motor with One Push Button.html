How It Works

This system uses a single push button to both start and stop a motor. Here's how it functions:

  1. Initial State: The motor is initially stopped.
  2. Starting the Motor: When the push button is pressed for the first time, it completes the circuit, allowing current to flow to the motor, which starts it.
  3. Keeping the Motor Running: A holding circuit or latching relay is used to keep the motor running even after the button is released.
  4. Stopping the Motor: When the push button is pressed again, it interrupts the holding circuit, cutting power to the motor and stopping it.

Components Required

  • Motor
  • Power supply
  • Push button (momentary switch)
  • Contactor or relay
  • Auxiliary contact
  • Overload protection device

Circuit Explanation

The circuit consists of the following key elements:

  1. Main Contact: This is the primary switch that controls power to the motor.
  2. Push Button: A normally open (NO) momentary switch that initiates both start and stop functions.
  3. Auxiliary Contact: This contact is mechanically linked to the main contact and is used in the control circuit.
  4. Holding Circuit: This circuit keeps the contactor energized after the push button is released.

Operation Sequence

  1. Starting: Pressing the button energizes the contactor coil, closing the main contact and starting the motor. The auxiliary contact also closes, creating a parallel path around the push button.
  2. Running: After releasing the button, the motor continues to run because the auxiliary contact maintains the circuit.
  3. Stopping: Pressing the button again interrupts the holding circuit, de-energizing the contactor and stopping the motor.

Safety Considerations

  • Always include proper overload protection for the motor.
  • Ensure all components are rated for the voltage and current of your system.
  • Include an emergency stop button in addition to the start/stop button for added safety.
  • Follow all local electrical codes and regulations when implementing this circuit.
